# K.B. Autoservis — preview

Preview web pro K.B. Autoservis Háj u Duchcova. Postavil [vt-digital.cz](https://vt-digital.cz).

## Stack
- Vanilla HTML5 + Tailwind CDN
- GSAP + ScrollTrigger (parallax, tilts)
- AOS (scroll animace)
- Swiper (recenze carousel)
- Static site, 0 build step

## Lokální preview
```
python3 -m http.server 5099
```
Otevři `http://localhost:5099/`
